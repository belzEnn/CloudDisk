### Step-by-Step Guide

1. **Clone the repository:**
    ```bash
    git clone https://github.com/belzEnn/DDDrive.git
    cd DDDrive
    ```
2. **Setup Virtual Environment:**
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    ```
3. **Install Dependencies:**
    ```
    pip install -r requirements.txt
    ```
4. **Enter your Telegram API key**
    - Go to [telegram.org](https://my.telegram.org/apps) and log in
    - Create a `.env` file and set two variables:
        ```
        API_ID=Your_API_ID_FROM_SITE
        API_HASH=Your_API_HASH_FROM_SITE
        ```
